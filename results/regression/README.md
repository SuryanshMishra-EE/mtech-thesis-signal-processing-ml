# Regression Results (MLR)

This folder contains results of the Multiple Linear Regression (MLR) model
evaluated using Leave-One-Subject-Out (LOSO) cross-validation.

## Model
- Regression type: Multiple Linear Regression
- Validation: LOSO
- Target: Lifted weight (kg)

## Key Results
- Mean Absolute Error (LOSO): ~0.36 kg
- Distinct weight levels: 4
