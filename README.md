# M.Tech Thesis – Signal Processing & Machine Learning

## Overview
This repository contains my M.Tech thesis work focused on signal processing and machine learning.  
The work covers data preprocessing, feature extraction, model development, and performance evaluation.

## Problem Statement
Develop a robust signal processing and machine learning pipeline for analyzing real-world signals, with emphasis on noise robustness, feature engineering, and classification performance.

## Methodology
- Signal preprocessing and filtering
- Feature extraction (time, frequency, and time–frequency domain)
- Classical machine learning models
- Deep learning–based approaches
- Model evaluation and comparison

## Tools & Technologies
- MATLAB
- Python
- NumPy, SciPy
- scikit-learn / PyTorch (as applicable)
